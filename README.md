# Gen-Password
---------------------------------
Программа для создания надежных паролей. Она создана для тех, кому надоело придумывать сложные комбинации из головы или пользоваться перегруженными онлайн-сервисами

A program for creating strong passwords. It is designed for those who are tired of coming up with complex combinations in their heads or using overloaded online services
